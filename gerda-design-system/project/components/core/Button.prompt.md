Editorial CTA button — wide uppercase tracking, near-square corners; use for hero CTAs, form submits, and gallery actions.

```jsx
<Button variant="line">View Gallery</Button>
<Button variant="solid" size="lg">Get in touch</Button>
<Button variant="ghost" as="a" href="#work">Selected work</Button>
```

Variants: `line` (underlined link, default), `solid` (filled dusty rose), `ghost` (hairline outline). Sizes: `sm` / `md` / `lg`. Render as a link with `as="a"`.
